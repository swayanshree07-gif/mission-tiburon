# tiburon-inductions

This repository contains implementations of ROS2-based robotics tasks including simulation, perception, communication, and Gazebo environments.

  - Shape Drawing
  - Navigation
  - GUI Services
  - Follow the Girl

💠 Perception
  - Buoyancy Slider
  - Multi-Bot Spawning

💠 ROS2 Communication
  - Frequency Control
  - Data Sync
    
💠 Gazebo Fortress
  - Edge Detection
  - Color Tracking
