# tiburon-inductions

## Overview
This repository contains implementations of ROS2-based robotics tasks including simulation, perception, communication, and Gazebo environments.

## Tasks Completed
💠 Turtle Simulation
  - Shape Drawing
  - Navigation
  - GUI Services
  - Follow the Girl

💠 Gazebo Ignition Fortress
  - Buoyancy Slider
  - Multi-Bot Spawning

💠 ROS2 Communication
  - Frequency Control
  - Data Sync
    
💠 Perception
  - Edge Detection
  - Color Tracking
